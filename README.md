## DRF API for trainers.

### Install.
Clone project:
```bash 
git clone https://github.com/AxmetES/DRF_apis_for_trainers.git
```
create docker image:
```bash
docker build -t my-django-app .
```

create and run container:
```bash
docker run -d -p 8000:8000 --name my-django-container my-django-app
```
### get start.
